## ✨ Features

- 🎨 **Tailwind CSS + DaisyUI** - 유틸리티 기반 스타일링
- 🌍 **Vue I18n** - 다국어 지원 (한국어/영어)
- 🔧 **MSW** - API 모킹으로 개발 효율성 향상
- 📚 **Storybook** - 컴포넌트 문서화 및 시각화
- 🚀 **GitHub Actions** - 자동 빌드 및 배포
- ⚡ **Vite** - 빠른 개발 환경
- 🟦 **TypeScript** - 타입 안전성

## 🛠️ Development

```bash
# 개발 서버 시작
npm run dev

# Storybook 실행
npm run storybook

# 빌드
npm run build

# 배포
npm run deploy
```
